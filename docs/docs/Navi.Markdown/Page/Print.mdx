# System.Boolean Print(System.String)

Writes the Markdown content of the page to the specified file path.
            Creates the directory if it does not exist.

**Parameters :** 

`System.String` path : The file path where the Markdown content will be saved.

**Returns :** `System.Boolean` : <c>true</c> if the file was written successfully; otherwise, <c>false</c>.
