# Page

Navi.Markdown.Page

Represents a Markdown page composed of multiple Markdown elements.

## Constructors

| Constructor              | Description                                                              |
| ------------------------ | ------------------------------------------------------------------------ |
| [Page()](./Constructors) | Initializes a new instance of the  class with an empty list of elements. |

## Methods

| Type             | Method                                  | Description                                                                                                                 |
| ---------------- | --------------------------------------- | --------------------------------------------------------------------------------------------------------------------------- |
| `System.Boolean` | [Print](../../Navi.Markdown/Page/Print) | Writes the Markdown content of the page to the specified file path.
            Creates the directory if it does not exist. |
