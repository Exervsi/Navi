# Page

Navi.Markdown.Page

Represents a Markdown page composed of multiple Markdown elements.

## Constructors

| Constructor              | Description                                                              |
| ------------------------ | ------------------------------------------------------------------------ |
| [Page()](./Constructors) | Initializes a new instance of the  class with an empty list of elements. |

## Properties

| Type     | Property | Description                                                            |
| -------- | -------- | ---------------------------------------------------------------------- |
| `List`1` | Elements | Gets or sets the list of Markdown elements that make up the page.      |
| `String` | Markdown | Gets the combined Markdown representation of all elements in the page. |

## Methods

| Type      | Method                                  | Description                                                                                                                |
| --------- | --------------------------------------- | -------------------------------------------------------------------------------------------------------------------------- |
| `Boolean` | [Print](../../Navi.Markdown/Page/Print) | Writes the Markdown content of the page to the specified file path.            Creates the directory if it does not exist. |
