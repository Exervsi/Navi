# Header

Navi.Markdown.Header

Represents a Markdown header element (e.g., # Header).

## Constructors

| Constructor                                 | Description                                                                 |
| ------------------------------------------- | --------------------------------------------------------------------------- |
| [Header(`String`, `Int32`)](./Constructors) | Initializes a new instance of the  class with the specified text and level. |

## Properties

| Type     | Property | Description                                                         |
| -------- | -------- | ------------------------------------------------------------------- |
| `String` | Text     | Gets or sets the text content of the header.                        |
| `Int32`  | Level    | Gets or sets the header level (1-6), where 1 is the largest header. |
| `String` | Markdown | Gets the Markdown representation of the header.                     |
