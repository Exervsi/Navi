# Navi.Markdown

## Types

| Type                                     | Description                                                                                                                                                                                               |
| ---------------------------------------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| [DocusaurusSidebar](./DocusaurusSidebar) | Represents a Docusaurus sidebar metadata block as a Markdown element.                                                                                                                                     |
| [Header](./Header)                       | Represents a Markdown header element (e.g., # Header).                                                                                                                                                    |
| [IMarkdownElement](./IMarkdownElement)   | Represents a generic element that can be converted to Markdown format.            Implementations of this interface should provide a Markdown representation            of the element via the  property. |
| [Page](./Page)                           | Represents a Markdown page composed of multiple Markdown elements.                                                                                                                                        |
| [Table](./Table)                         | Represents a Markdown table element with headers and values.                                                                                                                                              |
| [Tables](./Tables)                       | Provides static methods for generating and formatting Markdown tables from various data sources.                                                                                                          |
| [Text](./Text)                           | Represents a plain text element within a Markdown document.                                                                                                                                               |
