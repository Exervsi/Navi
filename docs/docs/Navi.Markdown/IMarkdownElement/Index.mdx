# IMarkdownElement

Navi.Markdown.IMarkdownElement

Represents a generic element that can be converted to Markdown format.
            Implementations of this interface should provide a Markdown representation
            of the element via the  property.

## Properties

| Type     | Property | Description                                      |
| -------- | -------- | ------------------------------------------------ |
| `String` | Markdown | Gets the Markdown representation of the element. |
