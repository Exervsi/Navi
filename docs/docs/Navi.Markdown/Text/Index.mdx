# Text

Navi.Markdown.Text

Represents a plain text element within a Markdown document.

## Constructors

| Constructor                      | Description                                                       |
| -------------------------------- | ----------------------------------------------------------------- |
| [Text(`String`)](./Constructors) | Initializes a new instance of the  class with the specified text. |

## Properties

| Type     | Property  | Description                                                             |
| -------- | --------- | ----------------------------------------------------------------------- |
| `String` | TextValue | Gets or sets the text value of this Markdown element.                   |
| `String` | Markdown  | Gets the Markdown representation of the text, followed by two newlines. |
