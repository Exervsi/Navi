# DocusaurusSidebar

Navi.Markdown.DocusaurusSidebar

Represents a Docusaurus sidebar metadata block as a Markdown element.

## Constructors

| Constructor                                            | Description                               |
| ------------------------------------------------------ | ----------------------------------------- |
| [DocusaurusSidebar(`String`, `Int32`)](./Constructors) | Initializes a new instance of the  class. |

## Properties

| Type     | Property | Description                                                          |
| -------- | -------- | -------------------------------------------------------------------- |
| `String` | Label    | Gets or sets the label to display in the Docusaurus sidebar.         |
| `Int32`  | Position | Gets or sets the position of the document in the Docusaurus sidebar. |
| `String` | Markdown | Gets the Markdown front matter block for the sidebar metadata.       |
