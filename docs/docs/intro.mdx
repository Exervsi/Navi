# Intro

Let's discover **Docusaurus in less than 5 minutes**.

## Getting Started

For now Navi can only make markdown trees and print out docusuaurus files.
Your code will look like this.

```chsarp
using Navi.InfoItems;

string dllPath = @"..\..\..\..\Navi\bin\Debug\netstandard2.0\Navi.dll";
string xmlPath = @"..\..\..\..\Navi\bin\Debug\netstandard2.0\Navi.xml";

DocuTree docuTree = new DocuTree(dllPath, xmlPath);
docuTree.PrintDocusaurus(@"..\..\..\..\..\docs\docs\");
```

So you only need to know three paths. The path to the .dll and .xml created with your project's build action, along with where you want the docusaurus files to save to.

That's it!
